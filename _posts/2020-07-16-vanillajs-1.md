## 1. What is Javascript?

- 웹에서 쓰이는 프로그래밍 언어

* JS: 중앙집중화 → 누군가 업데이트를 하면 모든 브라우저에서 작동
  <br>
  <br>

## 2. Specification (ECMAScript)

- 메뉴얼
  <br>

- 브라우저: 구글 크롬, 파이어폭스, 인터넷 익스플로러 → specification을 받아서 각자만의 방식으로 실행
- Specification의 버전: ES5(ECMAScript5), ES6(ECMAScript6)
  <br>
  <br>

## 3. Vanilla Javascript

- JS의 한 종류
  <br>

- 브라우저를 통해 우리에게 제공된 Javascript
- Library가 없음 → 날 것의 JS

  <br>

## 7. variable(변수)

- 라인 → expression 라고 부름
  <br>

- 모든 expression은 한 줄에 있어야 함
- expression이 끝날 때 **;** 를 붙임
  <br>
  <br>

- 초기화

  - **const(constant, 상수)**: 변수가 바뀌는 것 허용 X → 안정적
    <br>

  - **let**, **var(variance)**: 변수가 바뀌는 것 허용
  - <u>보통 const 사용</u> (정말 필요할 때만 let 사용)

```
# index.js

const a = 10;
let b = 10;
var c = 10;
```

  <br>

## 10. Arrays(정렬)
<h5>※ Camel case

 1. 변수명은 언제나 <u>소문자</u>로 시작한다.

1. 변수명 중간에 스페이스가 필요하면, <u>스페이스 대신 대문자</u>를 써주고 그 다음 다른 단어를 쓴다.</h5>


* array
<br>
ex. DB에서 가져온 list data

```

# index.js

const something = "Hello World!";

const dayOfWeek = ["Mon", "Tue", "Wed", "Thu", "Fri", "Sat", "Sun", 2020, true, something];

console.log(dayOfWeek);
console.log(dayOfWeek[0]);
console.log(dayOfWeek[5]);
console.log(dayOfWeek[9]);

```
