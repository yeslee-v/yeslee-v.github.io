## 2. Argument(인자)

- = parameter

- like variable
- ex. 계산기

```javascript
const Calculator = {
  plus: function (a, b) {
    return a + b;
  },
  minus: function (a, b) {
    return a - b;
  },
  multi: function (a, b) {
    return a * b;
  },
  divid: function (a, b) {
    return a / b;
  },
  square: function (a, b) {
    return a ** b;
  },
};
```
