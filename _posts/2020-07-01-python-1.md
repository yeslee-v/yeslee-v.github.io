다음과 같은 방법으로 풀어볼 수 있다.

```
a, b = map(int, input().split())

print(a // b, a % b)    # ①
print(*divmod(a, b))    # ②
```

연산 시간 비교해보았을 떄,

1. 작은 숫자일 때: ① > ②
2. 큰 숫자일 때: ② > ①

일수록 유리하다.

> 출처: 프로그래머스-파이썬은 파이썬답게
