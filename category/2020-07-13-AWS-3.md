## 1. S3

- Simple Storage Service

- 우리가 가지고 있는 파일을 보관해주는 서비스
- '파일 서버'로도 활용
- 버전 관리 기능

## 2. CRUD

- **C**reate
- **R**ead
- **U**pdate
- **D**elete

## 3. 3가지 구성요소

- Bucket
  - 1개의 버킷 = 1개의 프로젝트
- Folder
  - Bucket안에 Folder가 있음
- Object
  - Folder안에 파일(Object)이 있음
