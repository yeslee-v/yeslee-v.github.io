다음 두가지 방법으로 풀어볼 수 있다.

```
a, b = map(int, input().split())

print(a // b, a % b)    # ①
print(*divmod(a, b))    # ②
```

연산 시간을 비교해보았을 때,

1. **작은 숫자**일수록, <u>① 시간이 적게</U> 걸린다.
2. **큰 숫자**일수록, <u>② 시간이 적게</u> 걸린다.
